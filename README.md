# ⭐ CORRELATORS IN TERMS OF Single-HADRON OPERATORS ⭐


All single-hadron operators are taken from spectroscopy.pdf:

**Δ⁺⁺** = Deltapp,
**Δ̅⁺⁺** = DeltappB,
**Δ̅⁺** = DeltapB,
**Δ̅⁺** = DeltapB,
**Δ⁰** = Delta0,
**Δ̅⁰** = Delta0B,
**Δ⁻** = Deltam,
**Δ̅⁻** = DeltamB.

**N⁺** = Nucleonp,
**N̅⁺** = NucleonpB,
**N⁰** = Nucleon0,
**N̅⁰** = Nucleon0B.


**Σ⁺** = Sigmap,
**Σ̅⁺** = SigmapB,
**Σ⁰** = Sigma0,
**Σ̅⁰** = Sigma0B,
**Σ⁻** = Sigmam,
**Σ̅⁻** = SigmamB


**Ξ⁰** = Xi0,
**Ξ̅⁰** = Xi0B,
**Ξ⁻** = Xim0,
**Ξ̅⁻** = XimB.


**Λ⁰** = Lambda0,
**Λ̅⁰** = Lambda0B.
